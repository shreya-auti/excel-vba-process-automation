# Equipment Data Cleaning & Aggregation

## Purpose
This module demonstrates the data preparation and aggregation steps required before reliable automation and reporting can be applied.  
The dataset contained structural issues that needed to be resolved before meaningful analysis was possible.

## Work Performed
- Removed empty rows and duplicate records using filters and validation checks
- Corrected spelling errors and normalized text values
- Reconstructed broken department names using Flash Fill
- Validated data integrity using aggregate checks (SUM, AVERAGE, MIN, MAX, COUNT)
- Built multi-level pivot tables to summarize equipment counts by department and equipment class

## Outcome
Produced a clean, structured dataset and pivot-based summaries suitable for downstream automation, reporting, and decision support.

## Tools Used
- Microsoft Excel
- Filters & Conditional Formatting
- Flash Fill
- Pivot Tables
